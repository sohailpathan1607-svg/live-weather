# SkyCast Weather Forecast (Python Version)

This app is built using **Python (PyScript)** and **Tailwind CSS**.

## Deployment on GitHub Pages
1. Create a new repository on GitHub.
2. Upload ONLY the `index.html` file to the main folder of your repository.
3. Go to **Settings > Pages**.
4. Set the source to **Deploy from a branch** and select `main`.
5. Your site will be live at `https://your-username.github.io/your-repo-name/`.

## Why this works?
GitHub Pages only hosts static files (HTML/CSS/JS). By using **PyScript**, we can run Python logic directly in the browser without needing a Python server like Flask or Django.
